Это RUN Токен

RUN токен – это утилитарный внутренний токен приложения TheRun. В отличие от on-line игр в жанре фентэзи-аркада, TheRun – это уникальное приложение для соревнований в реальном мире с потенциалом в миллионы пользователей разных возрастов и категорий, которые будут активно использовать RUN токен

Получайте

Получайте RUN токены за участие в различных мероприятиях, таких как поединки, турниры и пр.

Приумножайте

Используйте RUN токены для стейкинга и получайте дополнительные RUN токены

Используйте

Покупайте на RUN токены билеты для участия в поединках/турнирах, NFT артефакты, аватарки и другое
